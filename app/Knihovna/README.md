# Knihovna - Matyas Balcar	

This is a readme for my library project, it also acts as a startup guide.


## Requirements

 - .NET 8 or newer
 - Firebird 5 or newer
 - isql (commes with firebird) to create db, you need isql.exe in the `DB SETUP` folder


## Setup

 1. Run `create_db.bat`. This creates the db and fills table with some test data. *(Knihovna.fbd)*
 2. Run the app.
